# Reference Links

This repository does not redistribute third-party PDFs, DBC databases, or sample logs.

Useful public references:

- ESP32RET: https://github.com/collin80/ESP32RET
- NimBLE-Arduino: https://github.com/h2zero/NimBLE-Arduino
- Arduino-ESP32: https://github.com/espressif/arduino-esp32
- CSS Electronics OBD2 DBC information: https://www.csselectronics.com/pages/obd2-dbc-file
- CSS Electronics OBD2 data pack information: https://www.csselectronics.com/pages/obd2-data-pack-car-dbc
- opendbc: https://github.com/commaai/opendbc

If you use any downloaded manuals, DBC files, or logs for private testing, keep them under `references/` locally. That directory is ignored by Git and should not be committed unless each file's redistribution license has been reviewed.
