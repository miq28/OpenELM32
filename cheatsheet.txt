```text id="pjlwmr"

```

Example:

```python id="k7jlwm"
import can

bus = can.interface.Bus(
    channel='can0',
    interface='socketcan'
)

print("ECU simulator started")

while True:
    msg = bus.recv()

    if msg is None:
        continue

    print(f"RX: {msg}")

    if msg.arbitration_id == 0x7DF:

        data = list(msg.data)

        # 0100
        if data[:3] == [0x02, 0x01, 0x00]:

            resp = can.Message(
                arbitration_id=0x7E8,
                data=[
                    0x06,
                    0x41,
                    0x00,
                    0xBE,
                    0x3F,
                    0xA8,
                    0x13,
                    0xAA
                ],
                is_extended_id=False
            )

            bus.send(resp)

            print(f"TX: {resp}")
```

Yes. Best repo I found for your exact car is:

```text
https://github.com/OBDb/Honda-CR-V
```

It is part of OBDb, which is an open community OBD database. The Honda CR-V repo says it contains signal set configurations by model year/version, intended for accurate signal mapping per CR-V generation. OBDb also has the standard SAE J1979 PID repo:

```text
https://github.com/OBDb/SAEJ1979
```

Use them like this:

- `OBDb/SAEJ1979`: standard OBD-II commands, safe baseline, Mode `01`, `09`, DTC reads, etc.
- `OBDb/Honda-CR-V`: Honda CR-V specific signal sets, likely where extended/enhanced PIDs live.
- Pelican’s Honda CR-V page confirms it has CR-V OBD PID support and lists 2013-2014 CR-V as a supported model-year range.

Important caution: extended/enhanced Honda PIDs are often not just simple `01xx`; they may use Mode `22`, custom headers, or module-specific addressing. Treat them as read-only first, and do not test anything that looks like clear/write/control: `04`, `14`, `2E`, `2F`, `31`, etc.

Sources:
- OBDb Honda CR-V repo: https://github.com/OBDb/Honda-CR-V
- OBDb SAE J1979 repo: https://github.com/OBDb/SAEJ1979
- OBDb organization: https://github.com/OBDb
- Pelican Honda CR-V OBD page: https://pelican.clutch.engineering/cars/honda/cr-v/
